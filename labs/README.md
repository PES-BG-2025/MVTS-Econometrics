# Laboratory Sessions

This folder contains R scripts and materials for hands-on laboratory sessions.

## Structure

Each lab folder contains:
- **R scripts**: Main lab exercises (e.g., `lab01_exercises.R`)
- **Instructions**: Lab guidelines and objectives (e.g., `lab01_instructions.pdf`)
- **Data**: Links or references to datasets in the `/data` folder

## Lab Organization

- `lab01/` - Introduction to R for Time Series
- `lab02/` - VAR Model Estimation and Diagnostics
- `lab03/` - Impulse Response Analysis
- `lab04/` - Cointegration Testing
- `lab05/` - VECM Implementation
- `lab06/` - Advanced Applications and Project Work

## Solutions

Solutions to lab exercises are available in the `/solutions` folder. Students are encouraged to attempt the exercises before consulting the solutions.

## Required R Packages

Common packages used throughout the labs:
- `vars` - VAR modeling
- `urca` - Unit root and cointegration testing
- `tseries` - Time series analysis
- `forecast` - Forecasting methods
- `ggplot2` - Visualization
- `dplyr` - Data manipulation
- `readr` - Data import
