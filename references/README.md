# References

This folder contains additional reading materials, papers, and reference documents.

## Organization

Materials can be organized by:
- Topic (e.g., `VAR_models/`, `cointegration/`)
- Type (e.g., `papers/`, `books/`, `tutorials/`)

## Content Types

- **Academic papers**: Research papers and journal articles
- **Book chapters**: Relevant textbook chapters or book excerpts
- **Technical notes**: Supplementary technical documentation
- **Tutorials**: Additional learning resources and guides
- **Software documentation**: Package documentation and vignettes
