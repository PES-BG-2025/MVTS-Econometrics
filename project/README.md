# Project

This folder contains materials related to the course project.

## Structure

This folder can include:
- **Project guidelines**: Instructions and requirements for the final project
- **Project data**: Datasets specifically for the project work
- **Templates**: R Markdown or R script templates for project submissions
- **Examples**: Sample projects or project proposals
- **Rubrics**: Evaluation criteria and grading rubrics

## Project Components

Students should organize their project work to include:
1. **Data exploration**: Initial analysis and visualization
2. **Model specification**: Choice and justification of models
3. **Implementation**: R code for model estimation
4. **Results**: Analysis and interpretation of findings
5. **Documentation**: Clear explanation of methodology and conclusions

## Submission Guidelines

Projects should be submitted as:
- R scripts (`.R` files) with well-commented code
- R Markdown documents (`.Rmd` files) for reproducible reports
- PDF reports generated from R Markdown

## Project Topics

Suggested topics for multivariate time series analysis:
- Macroeconomic forecasting (GDP, inflation, interest rates)
- Financial market analysis (stock returns, volatility)
- Energy market modeling (prices, consumption patterns)
- International trade dynamics
- Regional economic indicators

Students may propose their own topics subject to instructor approval.
