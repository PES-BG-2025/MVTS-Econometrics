# Lectures

This folder contains lecture slides and materials organized by week.

## Structure

Each week's folder can contain:
- **Slides**: PDF or PowerPoint presentations for lectures
- **Notes**: Additional lecture notes or handouts
- **Code examples**: R scripts demonstrating concepts covered in lectures

## Organization

- `week01/` - Introduction to Multivariate Time Series
- `week02/` - Vector Autoregression (VAR) Models
- `week03/` - Impulse Response Functions
- `week04/` - Forecast Error Variance Decomposition
- `week05/` - Cointegration and Error Correction Models
- `week06/` - Vector Error Correction Models (VECM)
- `week07/` - Structural VAR Models
- `week08/` - State Space Models
- `week09/` - Multivariate GARCH Models
- `week10/` - Factor Models and Dynamic Factor Models
- `week11/` - Regime Switching Models
- `week12/` - Advanced Topics and Applications

**Note**: Week topics are indicative and may be adjusted based on course progress.
